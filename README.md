# 🚀 AI Crypto Analyzer

一个免费的AI+Crypto项目评分工具，采用100分制评分框架，帮助用户快速评估AI+Crypto项目质量。

## ✨ 功能特性

- **100分制评分** - 技术(40分) + 经济(30分) + 市场(30分)
- **预设项目分析** - FET(82分), RNDR(78分), AGIX(71分), OCEAN(68分)
- **一键分享** - 自动生成X推文，方便传播
- **实时数据展示** - 模拟真实API数据效果
- **移动适配** - 全设备兼容，响应式设计
- **专业可视化** - 直观的评分图表展示

## 🎯 使用场景

1. **AI+Crypto项目研究** - 快速了解项目基本面
2. **投资决策辅助** - 数据驱动的分析框架
3. **内容创作工具** - 生成专业分析内容
4. **社群分享** - 一键分享项目评分结果

- **前端**: 纯HTML/CSS/JavaScript

- **部署**: GitHub Pages (免费)

## 📊 评分框架

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **技术基础** | 40分 | 技术创新、团队实力、开发活跃度 |
| **经济模型** | 30分 | 代币经济、激励机制、价值捕获 |
| **市场表现** | 30分 | 社区活跃、市场认知、成长潜力 |

## 🎮 使用方法

1. 输入项目符号（如：FET, RNDR, AGIX, OCEAN）
2. 点击"开始分析"按钮
3. 查看评分结果和详细分析
4. 点击"分享到X"分享结果
